Java 传值和传引用的区别

2018-11-05

前言

首先对传值和传引用要有个基本的概念

  • 传值：传递的是值的副本。方法中对副本的修改，不会影响到调用方。
  • 传引用：传递的是引用的副本，共用一个内存，会影响到调用方。此时，形参和实参指向同一个内存地址。对引用副本本身（对象地址）的修改，如设置为null，重新指向其他对象，不会影响到调用方。
